\# 🩺 Clinical AI Assistant (Healthcare LLM)



An AI-powered clinical assistant that analyzes medical notes and helps summarize patient information, identify diagnoses, and answer clinical questions.



This project demonstrates how Large Language Models (LLMs) can assist healthcare professionals by summarizing clinical documentation and retrieving relevant medical insights.



\---



\## 🚀 Features



\- Upload clinical reports (PDF)

\- Paste clinical notes

\- AI-powered summarization of patient information

\- Chat with a medical AI assistant

\- Retrieval-Augmented Generation (RAG) using FAISS

\- Clean Streamlit UI with chat-style interaction



\---



\## 🧠 Tech Stack



\- \*\*Python\*\*

\- \*\*Streamlit\*\* – Web interface

\- \*\*OpenAI API\*\* – LLM responses

\- \*\*FAISS\*\* – Vector similarity search

\- \*\*Sentence Transformers\*\* – Text embeddings

\- \*\*PyPDF\*\* – Extract text from PDFs
